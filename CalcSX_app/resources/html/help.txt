  <h3>Getting Started</h3>
  <ol>
    <li><b>Prepare your CSV</b> — columns must be labeled <b>x</b>, <b>y</b>, <b>z</b> (case-sensitive). If no headers are present, the first three columns are used. The curve does not need to be manually closed — CalcSX™ closes open loops automatically.</li>
    <li><b>Run the application</b> — <code>python -m CalcSX_app.main</code> from the repo root.</li>
    <li>Click <b>▲ Load CSV</b> in the ribbon FILE group — the coil wire appears in the viewport and "Coil" is added to the browser under <b>Coils</b>.</li>
    <li>Set coil parameters in the <b>Properties panel</b> (winds, current, tape dimensions, etc.).</li>
    <li>Click <b>▶ Run Analysis</b> — Forces, Stress, and B Axis layers appear; results summary populates.</li>
    <li>Switch to the <b>INSPECT</b> tab in the ribbon:
      <ul>
        <li><b>∿ Field Lines</b> — traces 3-D magnetic field lines. Set seed count in the Properties panel.</li>
        <li><b>⊡ Cross Section</b> — renders a 2-D B-field heatmap. Use <b>Section Pos. (m)</b> to slide the plane along the symmetry axis.</li>
      </ul>
    </li>
    <li>Toggle any layer using the eye icons (●/○) in the browser panel; legends hide and show with their layer.</li>
  </ol>

  <h3>Future Directions</h3>

  <b>High Priority</b>
  <ul>
    <li><b>GPU acceleration</b> — offload the Biot-Savart kernel to CUDA/Metal via CuPy or PyTorch so that large coil geometries (&gt;10 000 segments) and fine field-line integration become interactive-speed</li>
    <li><b>Real conductor cross-sections</b> — replace the filamentary wire model with a finite cross-section discretization (e.g., a stack of ReBCO tapes or a cable bundle); compute B and force distributions within the winding pack</li>
  </ul>

  <b>Physics Extensions</b>
  <ul>
    <li><b>Quench simulation</b> — thermal runaway model propagating through the winding pack; track normal-zone growth, hot-spot temperature, and energy dump into a protection circuit; visualize the propagating quench front in the 3-D viewport</li>
    <li><b>Field line topology tools</b> — Poincaré section maps, separatrix detection, and field-line connection-length plots for confinement analysis</li>
    <li><b>Multi-coil superposition</b> — load multiple CSVs and sum their B-fields; layer browser groups each coil independently; useful for solenoid pairs, Helmholtz coils, and tokamak PF-coil sets</li>
  </ul>

  <b>Design and CAD</b>
  <ul>
    <li><b>In-app coil prototyping</b> — parametric coil geometry builder (solenoid, saddle, helical, pancake) with real-time preview; export to CSV or STEP for manufacturing</li>
    <li><b>Click-to-inspect</b> — click any point on the coil wire to query local B, J×B, and stress; scroll the sidebar to the corresponding layer</li>
    <li><b>Undo / redo</b> — full action stack so CSV reloads and analysis re-runs can be stepped back</li>
  </ul>

  <b>Output and Integration</b>
  <ul>
    <li><b>VTK / ParaView export</b> — save computed field volumes and surface meshes for downstream post-processing</li>
    <li><b>2-D detail panels</b> — floating windows per layer with arc-length plots (force vs arc, stress vs arc, on-axis B-field profile)</li>
    <li><b>Persistent settings</b> — save last-used coil parameters, seed count, and section offset to a JSON config file</li>
    <li><b>Briefcase packaging</b> — signed, distributable macOS and Windows app bundles</li>
  </ul>

  <h3>Bobbin Import (.bobsx)</h3>
  <p>
    The <b>Import Bobbin</b> button accepts <code>.bobsx</code> files generated by the
    <b>CalcSX Fusion 360 Exporter</b> script (included in the <code>calcsx_exporter/</code> folder).
  </p>
  <p>
    <b>In Fusion 360:</b> open your bobbin model, go to UTILITIES &gt; Scripts and Add-Ins,
    run <code>calcsx_exporter</code>, select all groove floor faces, and export.
    Each face becomes one channel with exact surface normals from the CAD kernel.
  </p>
  <p>
    <b>In CalcSX:</b> click Import Bobbin, select the <code>.bobsx</code> file.
    Each channel appears as a separate coil — hide, delete, rename, or analyze
    individually.  Winding parameters (turns, current, tape dimensions) are set
    in the Properties panel after import.
  </p>
