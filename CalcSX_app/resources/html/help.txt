  <h3>What's New in v2.0.0</h3>

  <b>INSPECT Tools — Field Visualization Overhaul</b>
  <ul>
    <li><b>Magnetic field line tracing</b> — 3-D streamlines seeded on a Fibonacci sphere around the coil, integrated forward and backward via batched RK4; coloured by log₁₀|B| (cool colormap)</li>
    <li><b>Slidable cross-section plane</b> — 2-D B-field heatmap sliced perpendicular to the PCA symmetry axis; position adjustable via "Section Pos. (m)" in the Properties panel</li>
    <li><b>log₁₀ colormap scaling</b> — dynamic range compression so near-coil strong-field and far-field weak-field are simultaneously readable</li>
    <li><b>Independent scalar bars</b> — each INSPECT layer carries its own legend; visibility follows the eye-icon toggle in the browser</li>
    <li><b>Non-interfering layers</b> — Field Lines and Cross Section coexist; running one does not clear the other</li>
  </ul>

  <b>Workspace and UI</b>
  <ul>
    <li><b>Terrain camera mode</b> — camera up-axis locked to Z (Fusion 360-style); the coil appears to rotate above a fixed ground plane</li>
    <li><b>Improved floor grid</b> — higher contrast reference grid sized to 10× the coil diameter</li>
    <li><b>Zoom lock on normalize toggle</b> — toggling "Normalize Force Vectors" no longer resets the viewport camera</li>
  </ul>

  <h3>Getting Started</h3>
  <ol>
    <li><b>Prepare your CSV</b> — columns must be labeled <b>x</b>, <b>y</b>, <b>z</b> (case-sensitive). If no headers are present, the first three columns are used. The curve does not need to be manually closed; CalcSX™ closes open loops automatically.</li>
    <li><b>Run the application</b> — <code>python -m CalcSX_app.main</code> from the repo root.</li>
    <li>Click <b>▲ Load CSV</b> in the ribbon FILE group — the coil wire appears in the 3-D viewport and "Coil" is added to the browser under <b>Coils</b>.</li>
    <li>Set coil parameters in the <b>Properties panel</b>:
      <ul>
        <li><b>Winds</b> — total winding count</li>
        <li><b>Current (A)</b> — operating current</li>
        <li><b>Tape Thick (µm)</b> — per-layer tape thickness (default: 80 µm, standard ReBCO)</li>
        <li><b>Tape Width (mm)</b> — tape width (default: 4.00 mm)</li>
        <li><b>Axis Samples</b> — number of on-axis B-field profile points</li>
        <li><b>Gaussian Quadrature</b> — higher-accuracy force integration (avoid for &gt; ~300 pts)</li>
        <li><b>Normalize Force Vectors</b> — toggle between magnitude-encoded and uniform-length arrows</li>
      </ul>
    </li>
    <li>Click <b>▶ Run Analysis</b> — Forces, Stress, and B Axis layers appear; results summary populates.</li>
    <li>Switch to the <b>INSPECT</b> tab in the ribbon:
      <ul>
        <li><b>∿ Field Lines</b> — traces 3-D magnetic field lines. Set seed count in the Properties panel (8–60).</li>
        <li><b>⊡ Cross Section</b> — renders a 2-D B-field heatmap on a plane through the coil. Use <b>Section Pos. (m)</b> to slide the plane along the symmetry axis.</li>
      </ul>
    </li>
    <li>Toggle any layer using the eye icons (●/○) in the browser panel; legends hide and show with their layer.</li>
  </ol>

  <h3>Future Directions</h3>
  <ul>
    <li><b>GPU acceleration</b> — Biot-Savart kernel offloaded to CUDA/Metal for interactive-speed large coils</li>
    <li><b>Real conductor cross-sections</b> — finite cross-section winding pack (ReBCO tape stacks, cable bundles) with B and force distributions within the pack</li>
    <li><b>Quench simulation</b> — thermal runaway propagation, hot-spot temperature, energy dump, and animated quench-front visualization</li>
    <li><b>Multi-coil superposition</b> — load multiple CSVs with summed B-fields; independent browser groups per coil</li>
    <li><b>In-app coil prototyping</b> — parametric geometry builder (solenoid, saddle, helical, pancake) with real-time 3-D preview</li>
    <li><b>Field line topology</b> — Poincaré sections, separatrix detection, connection-length plots</li>
    <li><b>VTK / ParaView export</b> — save field volumes and surface meshes for downstream post-processing</li>
    <li><b>2-D detail panels</b> — floating arc-length plots (force, stress, on-axis B-field) per layer</li>
  </ul>
