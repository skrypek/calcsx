  <h3>What's New!</h3>
  <ul>
    <li><b>v1.1.0</b> — Performance &amp; usability overhaul</li>
    <ul>
      <li><b>Vectorized Biot-Savart engine</b>: all field evaluations now use fully NumPy-vectorized kernels — no Python loops over segments. Typical speedup: 10–50× for force integration, 100×+ for B-field cross-section</li>
      <li><b>Chunked cross-section sampling</b>: large grids computed in memory-safe batches; no RAM spikes for high-resolution runs</li>
      <li><b>Adaptive cross-section resolution</b>: new <i>Cross-section Grid</i> control (32–512 pts/axis) replaces the fixed 120×120 grid</li>
      <li><b>Configurable on-axis samples</b>: new <i>On-Axis Samples</i> control (50–1000 pts) for the B-field profile</li>
      <li><b>Save Results</b>: export arc-position, force density, hoop stress, and on-axis B-field to CSV from the results view</li>
      <li><b>Stage-aware loading screen</b>: progress dialog now shows the active computation step as a subtitle alongside the quippy messages</li>
    </ul>
    <li><b>v1.0.3</b> — functionality and bug fixes</li>
    <ul>
      <li>B-field cross section plotting</li>
      <li>Planar coil capability (beta)</li>
      <li>Force density calculations</li>
      <li>CalcSX™ watermark on plots</li>
    </ul>
  </ul>

  <h3>Getting Started</h3>
  <ol>
    <li><b>Load CSV → Select file</b>: Load a coil geometry, click <b>Preview Curve</b> to visualize the coordinates.</li>
      <ul>
      <li>CSV should have x, y, z column headers</li>
      </ul>
    <li>Enter magnet operating/manufacturing parameters: <b>Winds</b> and <b>Current</b>.</li>
    <li>Input tape parameters: <b>Thickness</b> and <b>Width</b>—system default is standard ReBCO.</li>
    <li>Decide on tasks to perform: check <b>Calculate B-Field Cross-section</b> for an additional on-plane calculation.</li>
    <ul>
     <li><b>WARNING</b>: computationally expensive</li>
     <li>System default is full coil Lorentz force density vector, hoop stress calculations, and on-axis B-field (these will be performed and displayed no matter user preference)</li>
    </ul>
    <li>Set B-field filter limit (cross section displays only regions under filter for higher fidelity simulation)</li>
    <li>Decide on computation method: check <b>Gaussian Quadrature</b> to enable higher accuracy force and stress modeling</li>
    <ul>
      <li><b>WARNING</b>: computationally expensive, not recommended for large samples (>300 points)</li>
      <li>System default is Simpson's Rule integration</li>
    </ul>
    <li>Click <b>Generate</b> to compute fields, forces, and stresses.</li>
  </ol>

  <h3>Interaction</h3>
  <ul>
    <li><b>3D Plots</b>:</li>
    <ol>
      <li><b>Rotate</b>: Left‑drag in the 3D view</li>
      <li><b>Normalize</b> Lorentz force density allows normalized vectors to be viewed. Simply check the <b>Show normalized vectors</b> box. This toggle preserves viewing angle and can be undone.</li>
      <li><b>Zooming</b>: Currently <b>NOT</b> a supported feature on any 3D plot generated by CalcSX™.</li>
    </ol>
    <li>2D Plots</b>:</li>
    <ol>
      <li><b>Cursor</b>: An interactive cursor is available on every 2D plot, enabling precise numerical analysis of the visual data.
      <li><b>Zooming</b>: Currently <b>NOT</b> a supported feature on any 2D plot generated by CalcSX™.</li>
  </ol>
  </ul>

  <h3>Features</h3>
  <ul>
    <li><b>Filament</b>: Parametric plotting of coil geometries with direction of curve procession, automatic axis of symmetry calculation via principal component analysis
    <li><b>B‑Field</b>: Magnitude calculations, on-axis B-field figure via principal component analysis, on-plane B-field heat map</li>
    <li><b>Lorentz Forces</b>: Segment‑wise J×B vector force density calculations utilizing Biot-Savart</li>
    <li><b>Stress</b>: Hoop stress (via membrane decomposition) approximations</li>
  </ul>
